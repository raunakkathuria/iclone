# inbox

Raw thoughts land here, one note per file (or appended lines). A line on a walk, a half-formed idea
in a meeting — capture it and move on. You do not process it now.

Later, the `process-inbox` skill reads the whole backlog, sorts each note (a post, a task, something
to remember, or nothing), drafts what should be drafted in your voice, and leaves it for you to
review. Processed notes move to `inbox/archive/`.

The capture is constant. The processing is scheduled. The publishing is gated — nothing leaves
without you saying so.
